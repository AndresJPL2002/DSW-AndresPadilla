Propósito

Se busca explicar de la manera mas clara posible, la temática de las etiquetas básicas de HTML, atributos y valores. Por lo que se puede decir que el propósito de este web es enseñar una parte de lo que es el lenguaje HTML, de manera que se comprenda y pueda solucionar sus dudas lo mejor posible.

Propósito del sitio

En este sitio web, se busca dar a conocer distintas temáticas relacionadas con el lenguaje HTML, y el desarrollo de paginas web. Esto se realiza con el fin de darle al lector información que pueda considerar útil con respecto a esta temática.
